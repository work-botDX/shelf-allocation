# プロット修正・本文設計計画

## Context

レビュー結果に基づき、plot-final.mdの修正と本文設計を行う。主な指摘事項は以下の通り：

1. ヴェルディアスの目的と手段の接続をより明確に
2. 7年後のエラの「孤独」の由来を伏線化
3. 誤字修正（「二度も」→「二度と」）
4. 第三章か第四章でもう一度、未来側から感情が漏れる場面を追加

---

## Phase 1: plot-final.md の修正

### 1.1 誤字修正
- 第9行：「二度も完全には切れない」→「二度と完全には切れない」

### 1.2 ヴェルディアスの目的と手段の接続強化
第三章のヴェルディアスの独白に以下を追加：
- 「魂の裂け目は聖剣の加護すら貫く綻び——世界の選別システムそのものを崩せる」
- これにより、エラ一人の悲劇が「世界規模の秩序破壊の起点」になることが明確になる

### 1.3 7年後のエラの感情漏れの追加
第四章の戦闘シーンに、もう一つ感情が漏れる場面を追加：
- 現在のエラが攻撃を受けた瞬間、7年後のエラが一瞬怯む
- 「......痛い、の......？」と漏らす
- これにより、同一魂の両側であることが感覚的に示される

### 1.4 7年後のエラの「孤独」の由来の伏線
第二章〜第三章で薄く伏線化：
- 第二章：禁術発動時、エラが一瞬「どこか遠くに意識が引っ張られる感覚」を感じる
- 第三章：7年後のエラが初登場時、虚ろな瞳で「......長かった」と呟く

## Phase 2: 本文設計

### 2.1 章ごとの文字数目安（長編版：30,000字以上）

| 章 | 目安文字数 | 主要感情 | 重要シーン |
|----|-----------|---------|-----------|
| 第一章 | 4,000〜5,000字 | 憧れ・安心 | 儀式・約束・聖剣選定 |
| 第二章 | 6,000〜7,000字 | 劣等感・恐怖 | 共闘・格差・禁術 |
| 第三章 | 5,000〜6,000字 | 不安・困惑 | 魔女登場・ヴェルディアス独白 |
| 第四章 | 5,000〜6,000字 | 衝撃・決断 | 真相開示・消滅連鎖 |
| 第五章 | 7,000〜8,000字 | 悲哀・受容 | 黒幕討伐・静止場面・未来を断つ |
| 結末 | 1,500〜2,000字 | 喪失・余韻 | 星見の花の丘 |

**合計目安**: 28,500〜34,000字（長編）

### 2.2 章ごとの感情設計詳細

#### 第一章：絆
- **感情の起点**: 幼い二人の無垢な信頼
- **感情の頂点**: 聖剣選定——アレンが選ばれ、エラが選ばれない
- **最後の余韻**: 「それでも、一緒に行く」というエラの決意

**本文のポイント**:
- 儀式の場面は「美しく、少し不思議」に
- 花が光る描写は控えめに——後の悲劇の伏線として
- 聖剣選定は「栄光」ではなく「分かれ道」として描く

#### 第二章：裂け目
- **感情の起点**: 共闘の高揚感
- **感情の転換**: 格差への苦しみ、ヴェルディアスの言葉
- **感情の頂点**: 禁術の選択——愛ゆえの「罪」

**本文のポイント**:
- 魂の接続の発露は「不思議だが心地よい」→「不気味」へ変化
- ヴェルディアスの言葉は「嘘ではないが、真実でもない」曖昧さを残す
- 禁術発動は「静かな決断」として描く——派手さより覚悟

#### 第三章：再来
- **感情の起点**: 日常への回帰の試み
- **感情の転換**: 魔女の登場——見覚えのある顔
- **感情の頂点**: ヴェルディアスの独白——世界の綻び

**本文のポイント**:
- 魔女の正体は「読者が気づく」→「キャラクターが気づく」の順で
- ヴェルディアスの独白は「説明的」になりすぎないよう、感情を混ぜる
- 彼の動機は「嫉妬」ではなく「存在への問い」として

#### 第四章：邂逅
- **感情の起点**: 戦闘の緊張
- **感情の転換**: 7年後のエラの一瞬の自我
- **感情の頂点**: 真相の開示——消滅連鎖

**本文のポイント**:
- 7年後のエラとの戦闘は「斬り合い」ではなく「押し合い」
- 「......私を、止めて」は「操られている中での懇願」ではなく「魂の底からの叫び」
- 真相は「説明」ではなく「エラの体感」として伝える

#### 第五章：約束
- **感情の起点**: 決戦の覚悟
- **感情の転換**: ヴェルディアスの死——しかし終わらない
- **感情の頂点**: 静止場面→未来を断つ

**本文のポイント**:
- ヴェルディアスの最期は「憎むべき敵」ではなく「同じ傷を持つ者」として
- 静止場面は「会話」ではなく「存在の共有」
- 未来を断つ一閃は「悲劇」ではなく「選択の完了」として描く

#### 結末
- **感情**: 喪失、しかし虚無ではない
- **余韻**: 「説明不能な何か」が残る

**本文のポイント**:
- アレンの視点のみ——エラはいない
- 記憶の欠落は「穴」ではなく「霧」として
- 最後の「ずっと、一緒——」は「約束の残響」

### 2.3 重要シーンの本文設計

#### シーンA：星見の花の下の儀式（第一章）
```
構成要素：
- 視覚：夜空、白い花びら、淡い光
- 聴覚：風の音、二人の声
- 触覚：手の温もり、花の茎

感情の流れ：
1. ワクワクする（儀式への期待）
2. 照れくさい（手をつなぐ）
3. 真剣になる（誓いの言葉）
4. 不思議な安心感（花が光る）

台詞のトーン：
- 幼いが、本気
- 「ずっと一緒」は「お願い」ではなく「確認」
```

#### シーンB：禁術の選択（第二章）
```
構成要素：
- 視覚：アレンの傷、血、エラの涙
- 聴覚：アレンの荒い息、エラの独り言
- 触覚：アレンの冷たくなる手

感情の流れ：
1. パニック（致命傷の瞬間）
2. 絶望（どうしようもない現実）
3. 思い出（禁術の知識）
4. 覚悟（選択）

台詞のトーン：
- エラは誰にも話しかけていない（独り言に近い）
- 「ごめんね」はアレンへではなく、未来の自分へ
```

#### シーンC：静止場面（第五章）
```
構成要素：
- 視覚：向き合う二人のエラ、夕暮れの光
- 聴覚：静寂、風の音
- 触覚：魂の亀裂の感覚

感情の流れ：
1. 静寂（戦闘の終わり）
2. 共鳴（魂の接続）
3. 理解（現在のエラが未来のエラの孤独を知る）
4. 受容（「私が終わらせる」という決断）

台詞のトーン：
- 7年後のエラの言葉は「呟き」
- 現在のエラは言葉を返さない（理解だけで十分）
```

#### シーンD：未来を断つ（第五章）
```
構成要素：
- 視覚：一閃、光の中に消える二人
- 聴覚：剣の音、アレンの叫び
- 触覚：魂が裂ける感覚

感情の流れ：
1. 決断（エラ）
2. 阻止しようとする（アレン）
3. 説得（エラ）
4. 完了（一閃）

台詞のトーン：
- エラは「謝罪」ではなく「説明」
- 「私が選ぶの」は「悲劇」ではなく「勝利」のトーンで
```

### 2.4 執筆時の注意事項

#### トーン・マナー
- 感傷的になりすぎない
- 「運命のいたずら」ではなく「選択の結果」として描く
- 説明は「対話」ではなく「体感」で伝える

#### 避けるべき表現
- 「まるで〜のようだ」の多用
- 感情を直接書く（「悲しかった」ではなく「胸が痛かった」）
- ご都合的なタイミングでの助け

#### リズム
- 戦闘シーン：短文で畳み掛ける
- 感情シーン：長文で深く描写
- 静止場面：余白を大切に
