## レビュー観点

### 1. 設定の整合性

- [ ] 魂ルールに矛盾がないか
  - 具現化は「干渉」ではなく「裂け目の固定化」として一貫しているか
  - ヴェルディアスが操作できる理由が明確か
  - 「外部干渉」と「自己切断」の差が論理的に説明されているか

- [ ] 消滅連鎖のルールが一貫しているか
  - どちらかを消せばもう片方も消える
  - 分離・封印・浄化は不可能
  - 外部干渉は消滅連鎖を加速させる

- [ ] 記憶喪失の因果がつながっているか
  - 儀式の副作用（接続が断たれると認識の輪郭も霧散する）
  - 結末での記憶喪失に説得力があるか

### 2. 伏線の配置

- [ ] 7年後エラの孤独の伏線が適切に配置されているか
  - 第二章：禁術発動時の「意識が引っ張られる感覚」
  - 第二章：裂けた未来側の意識は「断続的に夢のように存在」
  - 第三章：「......長かった」
  - 第四章：「......痛い、の......？」
  - 第五章：「......苦しかった。ずっと、一人で」

- [ ] ヴェルディアスの目的と手段の接続が明確か
  - 「魂の裂け目は聖剣の加護すら貫く綻び」
  - 世界規模の秩序破壊につながることが説明されているか

### 3. 感情設計

- [ ] エラの感情の流れが一貫しているか
  - 第二章：劣等感
  - 第三章：恐怖
  - 第四章：決断
  - 第五章：自己受容

- [ ] アレンの感情の流れが一貫しているか
  - 守りたい → 無力感 → 喪失

### 4. 章構成・分量

- [ ] 各章の文字数目安が適切か（長編：30,000字以上）
- [ ] 情報量の配分が適切か
- [ ] ペースが崩れないか

### 5. 商業性

- [ ] 4つの販売ポイントが明確か
  1. 幼い約束が悲劇の因果になる
  2. ヒロインが自分の未来を斬る
  3. 黒幕の思想がテーマと接続している
  4. 記憶ではなく喪失感だけが残るラスト

- [ ] ご都合主義的な解決がないか
- [ ] 黒幕の動機が単なる私怨に矮小化されていないか

### 6. 執筆ガイドの実用性

- [ ] 重要シーンの構成要素が具体的か
- [ ] トーン・マナーの指示が明確か
- [ ] 避けるべき表現が列挙されているか

### 7. 3ファイル間の整合性

- [ ] 同じ設定が3ファイル間で同じ粒度で反映されているか
  - 例：plot-final.mdの「具現化＝固定化」がplot-context.md、writing-guide.mdにも反映されているか
- [ ] 伏線の配置が3ファイル間で整合しているか
  - 例：7年後エラの孤独の伏線（第二章〜第五章）が全ファイルで一貫しているか
- [ ] 台詞や設定の重要な表現が3ファイル間で一致しているか
  - 例：「外部干渉」と「自己切断」の差が3ファイルで統一されているか
